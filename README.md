## System Nawadniania 

### Opis

System zarządzania podelwaniem ogrodu oparty na Raspberry Pi oraz Spring Boot.

Do stworzenia projektu wykorzystałem:

#### Soft
- Spring Boot
- Hibernate 
- Thymeleaf 
- Spring Security
- MySQL

#### Hardware
- Raspberry Pi 
- Przekaźniki
